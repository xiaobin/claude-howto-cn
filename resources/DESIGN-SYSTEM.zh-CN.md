# Claude How To - 设计系统

## 视觉标识

### 图标设计概念：带代码括号的指南针

Claude How To 图标使用**带 `>` 代码括号的指南针**来表示代码中的引导式导航：

```
     N（绿色）
     ▲
     │
W ───>─── E     指南针 = 引导/方向
     │          > 括号 = 代码/终端/CLI
     ▼
     S（黑色）
```

这创造了：
- **视觉清晰度**：立即传达"代码导航指南"
- **象征意义**：指南针 = 找到方向；`>` = 代码/终端
- **可扩展性**：可在 16px 到 512px 任意尺寸使用
- **品牌一致性**：与开发者工具美学相匹配，采用最小化配色

## 配色系统

### 调色板

| 颜色 | 十六进制 | RGB | 用途 |
|------|----------|-----|------|
| 黑色（主色） | `#000000` | 0, 0, 0 | 主描边、文本、南指针 |
| 白色（背景） | `#FFFFFF` | 255, 255, 255 | 浅色背景 |
| 灰色（次要） | `#6B7280` | 107, 114, 128 | 次要刻度线、次要文本 |
| 亮绿色（强调） | `#22C55E` | 34, 197, 94 | 北指针、圆点、强调线 |
| 近黑色（深色背景） | `#0A0A0A` | 10, 10, 10 | 深色模式背景 |

### 对比度（WCAG）

- 黑色在白色上：**21:1** AAA
- 灰色在白色上：**4.6:1** AA
- 绿色在白色上：**3.2:1**（仅装饰用，不用于文本）
- 白色在深色上：**19.5:1** AAA

### 强调色规则

**亮绿色（#22C55E）仅用于高亮：**
- 指南针北指针
- 圆心点
- 强调下划线/边框
- 禁止作为背景色
- 禁止用于正文文本

## 图标规格

### 指南针规格

指南针标识由以下几何元素构成：

```
元素                 | 描边/填充     | 颜色
--------------------|---------------|------------------
外环                | 3px 描边      | 黑色 / 白色（深色模式）
北刻度              | 2.5px 描边    | 黑色 / 白色（深色模式）
其他方位刻度        | 2px 描边      | 灰色 / 白色 50%（深色模式）
中间方位刻度        | 1.5px 描边    | 灰色 / 白色 40%（深色模式）
北指针              | 填充多边形    | #22C55E（始终绿色）
南指针              | 填充多边形    | 黑色 / 白色（深色模式）
> 括号              | 3px 描边      | 黑色 / 白色（深色模式）
圆心点              | 填充圆形      | #22C55E（始终绿色）
```

### 尺寸递进

```
16px  → 仅环 + 指针 + 箭头（极简）
32px  → 添加方位刻度线
64px  → 添加中间方位刻度线
128px → 完整细节，所有元素清晰
256px → 最大细节，粗描边
```

- **宽高比**：约 4.3:1（宽:高）

- **最小**：16px（网站图标）

- **最大**：无限制（矢量格式）

## 自定义

### 更改强调色

创建不同强调色变体：

1. 将所有 `#22C55E` 实例替换为你的强调色
2. 确保装饰元素对比度保持在 3:1 以上
3. 保持黑/白/灰结构不变

### 缩放

```css
svg {
  width: 256px;
  height: 256px;
}
```

SVG 通过 viewBox 自动缩放 — 无需变换。
