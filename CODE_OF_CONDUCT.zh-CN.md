# 贡献者行为准则

## 我们的承诺

我们致力于为所有人提供一个受欢迎且激励人心的社区。我们承诺，无论年龄、体型、种姓、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外表、种族、宗教或性认同和取向，参与我们社区都是每个人的无骚扰体验。

我们致力于提供一个积极、包容和安全的环境，让所有贡献者感到被尊重、重视和有能力贡献最佳工作。

## 我们的标准

有助于创造积极环境的行为示例包括：

### 保持尊重
- 使用受欢迎和包容的语言
- 尊重不同意见、观点和经历
- 优雅地接受建设性批评
- 尊重文化和语言差异
- 认可并庆祝多样化背景

### 保持协作
- 共同解决冲突
- 给予荣誉并认可他人贡献
- 帮助他人学习和成长
- 分享知识并提出问题
- 支持社区成员

### 保持专业
- 保持讨论专注和高效
- 避免贬低或侮辱性言论
- 在 issue 和讨论中保持主题
- 使用清晰和尊重的沟通方式
- 在他人的信息中假设善意

### 保持包容
- 欢迎新贡献者
- 帮助刚接触项目的人入门
- 提供建设性反馈
- 纳入不同观点的人
- 为安静的声音创造空间

## 不可接受的行为

以下行为在我们的社区中被视为骚扰，不可接受：

### 骚扰和歧视
- 与性别、性别认同和表达、性取向、残疾、精神疾病、神经（典型）性、身体外貌、体型、年龄、种族或宗教相关的攻击性评论
- 有关个人生活方式选择和实践的不受欢迎评论
- 故意用错性别或使用"死名"或被拒绝的名字
- 过度或离题的两性图片或行为
- 未经同意或在要求停止后进行身体接触和模拟身体接触

### 滥用行为
- 针对他人的暴力威胁或暴力语言
- 煽动对任何个人实施暴力的暴力语言
- 故意恐吓
- 跟踪或跟踪行为
- 通过重复联系进行骚扰
- 任何其他可能被合理认为不当的行为

### 在线骚扰
- 持续扰乱在线讨论
- 发布他人私人信息（人肉搜索）
- 不受欢迎的性关注或追求
- 恶意挑衅或故意破坏
- 垃圾信息或重复广告
- 传播虚假信息或捏造信息

### 报复
- 因报告违规而对某人采取报复行动
- 因某人报告不可接受行为而不公平对待
- 威胁要举报某人作为报复

## 范围

本行为准则适用于：
- 所有社区空间（GitHub issue、pull request、discussion 等）
- 项目仓库和文档
- 由项目组织或代表项目的活动和聚会
- 个人代表项目或社区的公共空间
- 与项目业务相关的私人通信

行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或社区时。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账户发帖，或在在线或离线活动中作为指定代表行事。

## 执行

社区负责人有责任澄清和执行可接受行为的标准，并将对任何被认定为不当、威胁、攻击性或有害的行为采取适当和公平的纠正措施。

### 报告违规行为

如果您经历或目睹了不可接受的行为，或有任何其他顾虑，请通过以下方式报告：

1. **电子邮件**：通过[提供联系方式（如有）]联系项目维护者
2. **GitHub Issues**：私密报告功能（如果在您的 GitHub 设置中可用）
3. **直接消息**：直接联系项目维护者

请提供：
- 您的联系方式
- 涉及人员的姓名（如果您愿意分享）
- 事件描述
- 背景和任何相关链接
- 事件如何影响您

**所有报告都将保密。**

### 调查和响应

社区负责人将：
- 及时公平地审查和调查所有投诉
- 对报告者保持保密
- 向报告者回复其报告的状态
- 根据调查结果采取适当行动

可能的行动包括：
- 私下警告或指导
- 公开道歉（如果违规者同意）
- 暂时或永久从社区空间移除
- 丧失贡献者权限
- 根据需要升级到 GitHub 或其他平台

### 上诉流程

如果您认为决定有误，可以通过以下方式上诉：
1. 提供新信息或上下文
2. 请求多名维护者审查
3. 在初步决定后 30 天内跟进

## 尊重分歧的准则

分歧在技术社区中是自然的。以下是尊重分歧的方式：

### 应该做
- 关注想法，而非个人
- 提出澄清性问题
- 承认对立观点中的有效点
- 使用"我"陈述（"我不同意，因为……"）
- 引用来源和证据
- 感谢他人的观点

### 不应该做
- 进行人身攻击
- 使用轻视性语言（"这很愚蠢"）
- 假设恶意
- 提起无关的过去冲突
- 升级语气或变得敌对
- 故意歪曲他人观点

### 尊重分歧的示例
```
错误："这是个愚蠢的想法，你显然不了解这个项目。"

正确："我理解你的观点，但我担心这种方法可能不适用于大型项目。你考虑过这个基准测试中显示的性能影响吗？我很想听听你对此的看法。"
```

## 建立桥梁

我们相信对话与和解的力量：

- 如果有人指出你的行为，倾听并反思
- 如果你造成了伤害，愿意道歉
- 支持他人在学习和成长
- 认识到人们会犯错
- 关注解决方案，而非责备
- 在保持问责的同时假设善意

## 遵循本准则的好处

通过维护尊重和包容的社区，我们：
- 吸引具有不同观点的多元化贡献者
- 通过协作创造更好的解决方案
- 建立更牢固的关系
- 使项目对新人更受欢迎
- 产生更高质量的文档和示例
- 培养学习和成长的文化

## 特别注意事项

### 对于维护者
- 在遵守本准则方面以身作则
- 对报告做出响应
- 对决定保持透明
- 公平地帮助解决冲突
- 为所有人创造安全环境

### 对于贡献者
- 假设维护者和其他贡献者有善意
- 提供建设性反馈
- 支持新人
- 帮助执行社区标准
- 对社区文化承担所有权

### 对于所有人
- 记住每条消息背后都有一个人
- 如果感到沮丧，请休息一下
- 如果需要支持，请联系社区负责人
- 帮助创建你想成为其中一员的社区
