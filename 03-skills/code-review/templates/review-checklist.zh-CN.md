# 代码审查清单

## 安全清单
- [ ] 无硬编码凭据或密钥
- [ ] 所有用户输入有输入验证
- [ ] SQL 注入防护（参数化查询）
- [ ] 状态变更操作有 CSRF 保护
- [ ] 正确的转义防止 XSS
- [ ] 受保护端点有身份验证检查
- [ ] 资源有授权检查
- [ ] 安全密码哈希（bcrypt、argon2）
- [ ] 日志中无敏感数据
- [ ] 强制使用 HTTPS

## 性能清单
- [ ] 无 N+1 查询
- [ ] 适当使用索引
- [ ] 有益处的地方实现了缓存
- [ ] 主线程上无阻塞操作
- [ ] 正确使用 async/await
- [ ] 大数据集有分页
- [ ] 数据库连接池化
- [ ] 正则表达式已优化
- [ ] 无不必要的对象创建
- [ ] 防止了内存泄漏

## 质量清单
- [ ] 函数少于 50 行
- [ ] 变量命名清晰
- [ ] 无重复代码
- [ ] 正确的错误处理
- [ ] 注释解释原因，而非内容
- [ ] 生产代码中无 console.log
- [ ] 有类型检查（TypeScript/JSDoc）
- [ ] 遵循 SOLID 原则
- [ ] 正确应用设计模式
- [ ] 代码自文档化

## 测试清单
- [ ] 编写了单元测试
- [ ] 覆盖了边界情况
- [ ] 测试了错误场景
- [ ] 有集成测试
- [ ] 覆盖率大于 80%
- [ ] 无 flaky 测试
- [ ] Mock 了外部依赖
- [ ] 测试名称清晰
